\doxysubsection{Opis szczegółowy}
Klasa \mbox{\hyperlink{class_organizm}{Organizm}}, która jest dziedziczona przez podklasy gatunków. 

Posiada odpowiednie implementacje czynności niezmiennych pomiędzy gatunkami oraz abstrakcyjne funkcje, które powinny być nadpisane przez podklasy gatunków dla czynności różniących się pomiędzy gatunkami.
